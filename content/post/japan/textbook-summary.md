### 📚 《大家的日语》初级1 语法知识点

| 课次 | 主要语法点 | 例句 |
| :--- | :--- | :--- |
| **第1课** | 1. **名词判断句**：～は～です (是…) <br>2. **否定形式**：～は～ではありません (不是…) <br>3. **疑问句**：～は～ですか (是…吗？) <br>4. **所属/同位**：～も～です (也是…) <br>5. **所属/同位**：～の～ (的/名词修饰)  | わたしは 学生です。(我是学生。)<br>かれは 学生ではありません。(他不是学生。)<br>あなたは 学生ですか。(你是学生吗？)<br>かれも 学生です。(他也是学生。)<br>わたしの 本です。(我的书。) |
| **第2课** | 1. **事物指示**：これ/それ/あれ (这个/那个/那个) <br>2. **疑问词**：どれ (哪个) <br>3. **名词链接**：～の～ (表示内容/所属)  | これは 本です。(这是书。)<br>それは 何ですか。(那是什么？)<br>あの 辞書は わたしのです。(那本词典是我的。) |
| **第3课** | 1. **地点指示**：ここ/そこ/あそこ (这里/那里/那里) <br>2. **疑问词**：どこ (哪里) <br>3. **价格**：～は いくらですか (多少钱) <br>4. **国家/产地**：～の くに/～せい (…的国家/…制造)  | ここは 教室です。(这里是教室。)<br>トイレは どこですか。(卫生间在哪里？)<br>これは いくらですか。(这个多少钱？)<br>これは 日本の 車です。(这是日本车。) |
| **第4课** | 1. **时间表达**：～時に (在…点) <br>2. **时段表达**：～から～まで (从…到…) <br>3. **动词ます形**：～ます (做…) <br>4. **否定形式**：～ません (不做…) <br>5. **过去形式**：～ました (做了…) <br>6. **过去否定**：～ませんでした (没做…)  | 7時に 起きます。(7点起床。)<br>9時から 5時まで 働きます。(从9点到5点工作。)<br>あした 休みます。(明天休息。) |
| **第5课** | 1. **方向助词**：～へ (去/往…) <br>2. **交通方式**：～で (乘坐…) <br>3. **同伴**：～と (和…一起) <br>4. **移动动词**：行きます/来ます/帰ります (去/来/回) <br>5. **疑问词**：いつ (什么时候)  | 日本へ 行きます。(去日本。)<br>バスで 学校へ 行きます。(坐公交去学校。)<br>友達と 帰ります。(和朋友一起回去。) |
| **第6课** | 1. **动作地点**：～で ～ます (在…做…) <br>2. **动作对象**：～を ～ます (做…/把…) <br>3. **邀请/提议**：～ませんか/～ましょう (不…吗？/做…吧) <br>4. **动作顺序**：～てから (…之后)  | 図書館で 勉強します。(在图书馆学习。)<br>コーヒーを 飲みます。(喝咖啡。)<br>一緒に 行きませんか。(一起去好吗？)<br>映画を 見ましょう。(看电影吧。) |
| **第7课** | 1. **方法/手段**：～で (用…) <br>2. **授受动词**：あげます (给)/もらいます (得到) <br>3. **动作对象**：～に あげます/もらいます (给…/从…得到)  | はしで 食べます。(用筷子吃。)<br>わたしは 山田さんに 花を あげます。(我给山田花。)<br>わたしは 山田さんに 花を もらいます。(我从山田那得到花。) |
| **第8课** | 1. **形容词/形容动词现在肯定**：～いです/～です (是…的) <br>2. **现在否定**：～くないです/～ではありません (不…/不是…) <br>3. **过去形式**：～かったです/～でした (曾经…/是…的) <br>4. **过去否定**：～くなかったです/～ではありませんでした <br>5. **陈述理由**：～が (表示转折/顺接)  | この 料理は おいしいです。(这道菜很好吃。)<br>この 町は にぎやかです。(这个城市很热闹。)<br>きのうは さむくなかったです。(昨天不冷。) |
| **第9课** | 1. **能力/喜好**：～が できます/好きです/上手です (会…/喜欢…/擅长…) <br>2. **理由说明**：～から (因为…) <br>3. **存在文**：～に ～が あります/います (在…有…)  | わたしは 日本語が わかります。(我懂日语。)<br>あそこに コンビニが あります。(那里有便利店。) |
| **第10课** | 1. **存在句**：～は ～に あります/います (位于…) <br>2. **位置关系词**：うえ/した/まえ/うしろ/なか/そと/となり/ちかく (上/下/前/后/里/外/旁边/附近) <br>3. **列举**：～や～など (…和…等)  | 本は つくえの うえに あります。(书在桌子上。)<br>部屋に ベッドや テーブルなどが あります。(房间里有床、桌子等。) |
| **第11课** | 1. **数量词**：位置、询问方式、概数 (使用「くらい」)。表示时间的长度用「どのくらい」。 | りんごを 三つ ください。(请给我三个苹果。)<br>この かいぎは 二時間ぐらい かかります。(这个会议要开两个小时左右。) |
| **第12课** | 1. **形容词/名词过去式**：でした/かったです<br>2. **比较**：～より～の ほうが～ (A比B更…)<br>3. **最高级**：～の なかで～が いちばん～ (在…中，…最…) | きのうは 雨でした。(昨天下雨。)<br>日本より 中国の ほうが 広いです。(和日本相比，中国更大。)<br>クラスで 李さんが いちばん 高いです。(班里小李最高。) |
| **第13课** | 1. **表达愿望**：～が ほしいです (想要…) / ～たいです (想做…)<br>2. **目的**：～に 行きます/来ます (去/来做…) <br>3. **描述**：形容词/形容动词修饰名词 | 新しい 車が ほしいです。(想要新车。)<br>日本へ 旅行に 行きたいです。(想去日本旅行。)<br>これは おいしい ケーキです。(这是美味的蛋糕。) |
| **第14课** | 1. **动词て形**<br>2. **请求**：～てください (请…)<br>3. **正在进行**：～ています (正在…)<br>4. **连接**：～て、～ (连接动词或句子)<br>5. **状态**：～ています (表示状态的持续，如「結婚しています」) | ちょっと 待ってください。(请等一下。)<br>今、新聞を 読んでいます。(现在正在看报纸。)<br>朝ごはんを 食べて、学校へ 行きます。(吃完早饭去学校。) |
| **第15课** | 1. **许可**：～ても いいです (可以…)<br>2. **禁止**：～ては いけません (不能…)<br>3. **动作完成/结果状态**：～ています (如「もう 帰りました」) | 写真を とっても いいですか。(可以拍照吗？)<br>ここで たばこを すっては いけません。(不能在这里吸烟。) |
| **第16课** | 1. **动作先后**：～てから (…之后)<br>2. **客观变化**：～くなります/になります (变得…)<br>3. **主观改变**：～くします/にします (使之变得…) | 仕事が 終わってから、飲みに 行きます。(工作结束后去喝酒。)<br>これから だんだん さむくなります。(今后会逐渐变冷。) |
| **第17课** | 1. **动词ない形 (否定形)**<br>2. **否定请求**：～ないでください (请不要…)<br>3. **义务/必须**：～なければ なりません (必须…)<br>4. **许可（否定）**：～なくても いいです (不…也可以) | ここで 写真を とらないでください。(请不要在这里拍照。)<br>レポートを 出さなければ なりません。(必须提交报告。) |
| **第18课** | 1. **动词辞书形 (原形)**<br>2. **能力/可能/爱好**：辞书形 + ことが できます/趣味です (能做…/爱好是…) <br>3. **动作前**：辞书形 + まえに (在…之前) | わたしは 日本語を 話すことが できます。(我会说日语。)<br>毎朝、朝ごはんを 食べるまえに、コーヒーを 飲みます。(每天早上，吃饭之前会喝咖啡。) |
| **第19课** | 1. **动词た形**<br>2. **经历**：～たことが あります (有过…经验)<br>3. **建议/劝告**：～たり、～たりします (又…又…/时而…时而…) <br>4. **客观变化**：～く/に なります (变得…) | 富士山に のぼったことが あります。(爬过富士山。)<br>日曜日は、本を 読んだり、テレビを 見たりします。(星期天，有时看看书，有时看看电视。) |
| **第20课** | 1. **简体形**（普通形）：动词/形容词/名词/形容动词的简体现在/过去，肯定/否定形式 | 明日、映画を 見る？(明天看电影吗？) <br>うん、見る。(嗯，看。) |
| **第21课** | 1. **简体句 + と思う**：表达个人想法和判断<br>2. **简体句 + と言う**：表示引用「～と言います」<br>3. **发生/举行**：名词(地点) + で + 名词が あります | わたしは 明日 雨が ふると 思います。(我想明天会下雨。) <br>田中さんは 来週 休みたいと 言いました。(田中说下周想休息。)<br>東京で オリンピックが あります。(东京将举办奥运会。) |
| **第22课** | 1. **连体修饰**：用简体句修饰名词 | これは 母が 作った ケーキです。(这是妈妈做的蛋糕。) <br>あそこに いる 人は 李さんです。(在那边的那个人是小李。) |
| **第23课** | 1. **条件/契机**：～と (一…就…) / ～とき (…的时候) | この ボタンを おすと、おつりが 出ます。(一按这个按钮，零钱就会出来。)<br>暇な とき、本を 読みます。(有空的时候看书。) |
| **第24课** | 1. **授受动词的用法区分**：<br>   - あげる (我方给出去)<br>   - もらう (我方得到)<br>   - くれる (别人给我或我方) | わたしは 山田さんに 花を あげました。(我把花给了山田。)<br>山田さんが わたしに 花を くれました。(山田把花给了我。) |
| **第25课** | 1. **假定条件**：～たら (如果…)<br>2. **顺接条件**：～ても (即使…也…) | もし お金が あったら、旅行に 行きたいです。(如果有钱，想去旅行。)<br>雨が ふっても、出かけます。(即使下雨，也要出门。) |
